## Hadoop, MapReduce和Spark-都是些啥？

Hadoop是一个开源技术集，促进高分布式处理和大数据集存储

纠其核心，Hadoop项目本身仅由几个组件组成。你可以将以下组件视为Hadoop完成高分布式处理的"Hadoop 魔法"方式
  - Hadoop Common
  - HDFS (Hadoop Distributed File System)
以及下面使用MapReduce或其他编程模型为Hadoop构建数据处理程序的方式
  - Hadoop YARN
  - Hadoop MapReduce
